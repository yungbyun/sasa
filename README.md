## 공학사사 2023
### Introduction
* 본 사이트는 2023년 7월에 제주대학교에서 열린 공학사사 프로젝트 일부로 "인공지능으로 배우는 데이터 과학, 똑똑한 컴퓨팅"을 활동 주제로 하여 참여한 6개 팀의 활동을 위한 사이트입니다.
